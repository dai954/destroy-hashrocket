# このアプリについて DestroyHashRocket
 ハッシュロケット型を使用したコードを自動でシンボル型に変換してくれます。

# 使い方
URL <https://destroy-hashrocket.herokuapp.com>  
変換したいコードを左の枠内にコピー&ペーストします。
